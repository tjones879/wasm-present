What is WASM?

Web assembly serves as a new compilation target for web developers. This standard is being developed by major players including Apple, Google, Microsot, and Mozilla.

Who uses it, and is it stable?

Currently, web assembly is not yet used. There are multiple issues preventing widespread adoption, namely:

  1. The standard is still rapidly changing.
  2. Browser compatibility is lacking.
  3. Source languages are few.
